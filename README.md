# 🌌 Cat's Cradle Neon Hand Tracker

An immersive hand-tracking visualization inspired by the classic **Cat's Cradle** string pattern. Using **MediaPipe Hands** and the HTML5 Canvas API, this project creates glowing neon connections between both hands in real-time, producing a futuristic interactive web experience.

## ✨ Features

* 🖐️ Real-time hand tracking using MediaPipe
* 👋 Supports tracking up to 2 hands simultaneously
* 🌈 Neon-colored fingertip highlights
* 🕸️ Dynamic Cat's Cradle style string connections
* 💫 Glowing particle burst effects on fingertip touch
* 🎨 Cyberpunk-inspired visual design
* ⚡ Smooth canvas-based rendering
* 📷 Webcam-powered interaction
* 📱 Responsive fullscreen experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* MediaPipe Hands
* MediaPipe Camera Utils
* HTML5 Canvas API

## 🎮 Controls

| Action                    | Result                            |
| ------------------------- | --------------------------------- |
| Show Hands to Camera      | Track Hand Skeleton               |
| Move Fingers              | Generate Neon Web Patterns        |
| Touch Matching Fingertips | Create Particle Burst Effect      |
| Use Both Hands            | Activate Cat's Cradle Connections |
